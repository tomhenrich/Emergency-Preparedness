# Emergency kit packing lists

## Personal evacuation bags (each)
- [ ] One set of clothes
  - [ ] Pants, shirt, socks + underwear
- [ ] Toiletries/hygiene
  - [ ] Toothbrush + toothpaste
  - [ ] Deodorant
  - [ ] Hand sanitizer
  - [ ] Wet wipes
  - [ ] Toilet paper roll
  - [ ] Waste bags roll
  - [ ] Gold bond/anti-chafe
- [ ] Water bottle (Nalgene)
  - [ ] Electrolyte powder packets
- [ ] Charging (in mesh case)
  - [ ] 30W USB-C charger block
  - [ ] USB-C to USB-micro cable
  - [ ] USB-C to USB-C cable
  - [ ] **Add portable battery when leaving**
- [ ] Flashlight + AA Batteries (3 in baggie)
- [ ] Breathing
  - [ ] N95 masks (2)
  - [ ] Respirator half-face mask
  - [ ] Respirator P100 filters (2 pair)
- [ ] First-aid kit
- [ ] Notebook & sharpie
- [ ] Whistle
- [ ] Bic lighter
- [ ] Foil blanket
- [ ] Hand warmers (3)
- [ ] P-38 can opener (in plastic case)
- [ ] Work gloves (1 pair)
- [ ] Foam ear plugs
- [ ] Trash bag (1)
- [ ] Drawstring bag (1)
